CA1

For CA1, I already had experience making frontend websites in my first year of Polytechnic as I am in the Diploma in Information Technology Course.
Thus, using my previous knowledge, I decided to create this website using Bootstrap 5 to make styling the website faster and nicer.
Default navbar that comes with bootstrap
Sections I made
I started by using the default made navbar from bootstrap. I then used section and flex boxes to make sections that I can copy paste to add content into my website.
I split the website into 3 parts, the index and about pages, pages about my lessons and pages about my assignments.
For the lesson and CA1 pages, I would use bootstrap's carousel feature to neatly display multiple images to detail my journey.

Fusion

Tinkercad
